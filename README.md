# honki-base

### Setup

```
npm i
```

## GitHub Pages settings

Settings > Pages を選択します。

- Source
  - Deploy from branch
- Branch
  - main
  - /doc

で設定して保存します。

## build

```
npm run build
```

ビルドします。

## 初コミット

コミットします。

## serve

ローカルで確認したいときは以下のコマンドを実行して http://localhost:4000/ で表示します。

```
npm run serve
```
